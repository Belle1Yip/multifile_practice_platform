## 1. 平台概述

本平台是一个综合性的编程学习在线评测平台，主要特点包括：

- **多模式支持**：支持 LeetCode 模式、多文件模式、ACM 模式
- **AI 辅助**：内置 AI 大模型，自动生成测试用例
- **灵活判题**：支持 ACM 模式和 OI 模式打分
- **题库管理**：支持创建、编辑、删除题目
- **组卷功能**：自定义组卷，支持考试管理
- **题库导入**：支持从主流 OJ 平台导入题目

## 2. 快速开始

### 2.1 环境要求

- Node.js 20.x 或更高版本
- MongoDB 7.x 或更高版本
- 现代浏览器（Chrome、Firefox、Safari、Edge）

### 2.2 启动平台

```bash
# 启动 MongoDB（如果使用 Docker）
docker run -d -p 27017:27017 --name mongo mongo:7

# 启动后端服务
cd backend
npm install
npm run dev

# 启动前端服务（新终端）
cd frontend
npm install
npm run dev
```

平台启动后访问 http://localhost:5173 即可使用。

## 3. 用户注册与登录

### 3.1 注册

1. 点击页面右上角的「注册」按钮
2. 填写用户名、邮箱、密码
3. 确认密码后点击「注册」

### 3.2 登录

1. 点击页面右上角的「登录」按钮
2. 输入邮箱和密码
3. 点击「登录」

### 3.3 角色说明

| 角色 | 权限 |
|-----|------|
| 普通用户 | 浏览题目、提交代码、查看提交记录 |
| 管理员 | 管理题库、创建试卷、导入题目 |

## 4. 题库使用

### 4.1 浏览题目

1. 点击导航栏的「题库」菜单
2. 可以按难度、类型筛选题目
3. 使用搜索框搜索题目

### 4.2 查看题目详情

1. 在题库列表中点击题目名称或「开始做题」按钮
2. 页面左侧显示题目描述、输入输出格式、样例
3. 页面右侧显示代码编辑器

---

## 5. 题目提交与判题

### 5.1 编写代码

1. 选择编程语言（支持 Python、JavaScript、TypeScript、C++、Java、C）
2. 在代码编辑器中编写代码
3. 可以使用模板代码作为起点

### 5.2 选择判题模式

- **ACM 模式**：答案完全正确得满分，错误得 0 分
- **OI 模式**：按测试点得分累加，部分正确部分得分

### 5.3 提交代码

1. 点击「提交代码」按钮
2. 等待判题结果（通常几秒钟）
3. 查看判题结果和测试点详情

### 5.4 查看提交记录

1. 在题目详情页点击「提交记录」标签
2. 查看历史提交记录和结果

## 6. 创建题目（管理员）

### 6.1 进入创建页面

1. 登录管理员账户
2. 在题库页面点击「创建题目」按钮

### 6.2 填写题目信息

| 字段 | 说明 |
|-----|------|
| 题目标题 | 题目的名称 |
| 题目类型 | LeetCode / 多文件 / ACM |
| 难度 | 简单 / 中等 / 困难 |
| 标签 | 用于分类和搜索 |
| 题目描述 | 题目的详细描述 |
| 输入格式 | 输入数据的格式说明 |
| 输出格式 | 输出数据的格式说明 |

### 6.3 添加测试用例

1. 点击「添加」按钮添加测试点
2. 填写输入、输出和分值
3. 可以使用 AI 自动生成测试用例

### 6.4 添加模板代码

1. 选择编程语言
2. 输入该语言的模板代码
3. 支持为多种语言添加模板

## 7. AI 测试点生成

### 7.1 使用场景

当创建题目时，可以使用 AI 自动生成测试用例：

1. 填写题目描述、输入输出格式
2. 添加至少一个样例作为参考
3. 点击「AI生成」按钮
4. AI 会根据题目描述生成 5 个测试用例

### 7.2 注意事项

- 需要填写完整的题目描述才能生成有效测试用例
- AI 生成的测试用例需要人工审核
- 可以多次生成，选择合适的测试用例

## 8. 组卷功能

### 8.1 创建试卷

1. 点击导航栏的「组卷」菜单
2. 点击「创建试卷」按钮
3. 填写试卷信息：
   - 试卷标题
   - 试卷描述（可选）
   - 开始时间和结束时间
   - 判题模式（ACM/OI）

### 8.2 添加题目

1. 从下拉菜单选择题目
2. 设置题目分值
3. 点击「添加」按钮
4. 可以调整题目顺序和分值

### 8.3 管理试卷

- 查看试卷列表
- 编辑试卷信息
- 删除试卷

## 9. 题库导入

### 9.1 从 OJ 平台导入

目前支持从以下平台导入：

- LeetCode
- Codeforces

**操作步骤：**

1. 登录管理员账户
2. 进入创建题目页面
3. 在导入区域输入 OJ 题目链接
4. 点击「导入」按钮

### 9.2 从 PDF 导入

**操作步骤：**

1. 准备包含题目的 PDF 文件
2. 在导入页面选择 PDF 文件
3. 点击「上传并解析」
4. 系统会自动解析并创建题目

## 10. 判题模式说明

### 10.1 ACM 模式

- **评分规则**：所有测试点全部通过得满分，否则得 0 分
- **适用场景**：编程竞赛、面试题

### 10.2 OI 模式

- **评分规则**：按测试点得分累加，每个测试点有独立分值
- **适用场景**：算法竞赛、信息学竞赛

### 10.3 测试点状态

| 状态 | 说明 |
|-----|------|
| 通过 | 答案正确 |
| 答案错误 | 输出与预期不符 |
| 运行时错误 | 程序运行时出错 |
| 超时 | 超出时间限制 |
| 内存超限 | 超出内存限制 |
| 编译错误 | 代码编译失败 |

## 11. 常见问题

### Q1：如何成为管理员？

A：默认注册的用户为普通用户，需要管理员手动在数据库中修改用户角色。

### Q2：支持哪些编程语言？

A：目前支持 Python、JavaScript、TypeScript、C++、Java、C。

### Q3：判题需要多长时间？

A：通常在几秒钟内完成，复杂题目可能需要更长时间。

### Q4：如何导入大量题目？

A：可以通过以下方式批量导入：
- 使用 OJ 链接导入
- 使用 PDF 文件导入

### Q5：代码执行环境安全吗？

A：是的，所有代码在隔离的沙箱环境中执行，确保安全性。
